# QA & Associates Website

Premium one-page React + Vite + Tailwind website for QA & Associates, GST & Income Tax Consultants in Mendhar, Poonch, Jammu & Kashmir.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Lucide React icons

## Installation

```bash
npm install
npm run dev
```

Open the local URL shown by Vite.

## Production Build

```bash
npm run build
npm run preview
```

## Deploy on Vercel

1. Push this folder to a GitHub repository.
2. Import the repository in Vercel.
3. Use the default Vite settings:
   - Build Command: `npm run build`
   - Output Directory: `dist`
4. Deploy.
